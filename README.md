# cybozu-office-10-fix 

サイボウズ Office 10.0～10.5をGoogle Chromeから開いたとき、
「HTTP 200エラー」と表示される不具合を修正します。

この不具合は最新のサイボウズ Office 10.xでは修正されています。
ライセンス期限が終了し、最新版に更新できない場合のために作成しました。

# Install

Google Web Store からインストールできます。
